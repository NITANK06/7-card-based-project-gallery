# Card-Based Project Gallery

 Project Overview

 
This project showcases a gallery of projects using **card components**.  
Each card includes:
 An image
 A title
 A short description
 Hover effects with smooth scaling and shadow animations



 Features

 
 **Responsive grid layout** using CSS Grid
 **Card design** with rounded corners and subtle shadows
 **Hover animation**: scale up + shadow depth for interactive feel
 **Smooth transitions** for polished user experience
 At least **6 project cards** included
